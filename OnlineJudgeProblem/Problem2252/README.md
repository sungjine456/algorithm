## Problem : 2252(줄 세우기)

---
### 풀이 : 예제 입력2의 출력과는 출력 결과는 다르나 정답이다.
#### map을 통해서 간선을 모두 저장하고 int[]을 통해서 앞에 나와야 하는 수의 개수를 담는다.
#### 시작은 arr[]이 0인 인덱스를 queue에 담고 queue가 비어 있지 않으면 while문을 돈다.
#### queue에 담긴 값을 차례대로 poll하여 stringBuilder에 담고 poll한 값과 연결된 값들을 map에서 찾는다.
#### map에서 나온 값들의 arr[]값에 1을 뺀 후 0인지 비교하여 다음에 출력되어야 하는 값인지 확인하여 0이라면 queue에 담는다.
#### 위 내용을 queue가 빌 때 까지 반복한다.
