## Problem : 2606(바이러스)

---
### 풀이 : 
#### 이중배열을 만들어 입력받는 숫자들의 배열위치에 '1'을 넣는다.
#### 이 문제같은 경우는 양방향이기 때문에 한 쌍에 두가지 경우로 배열에 '1'을 넣아야 된다.
#### 배열에 '1'을 모두 넣은 후 비교를 시작하는데 바이러스는 1부터 시작하기 때문에 1부터 시작한다.
#### 바이러스에 걸렸었는지 먼저 확인한 후 이중 배열에 '1'이 담겨있는지 확인한다.
#### 만약 '1'이 있다면 바이러스에 걸리는 경우이므로 바이러스 채크를 해주고 카운트를 새준다.
